    使用git工具
    1、先在线上创建项目
    2、打开git工具 克隆 clone  项目地址 右键Paste粘贴
        git clone 项目地址
    3、要把 .gitignore 和 readme.md 两个文件放到项目里面去。。
    .gitignore文件的作用 - 里面写的目录都不会被提交上来。。。
    
    4、上传代码
        git add .
        git commit -m ......
        git pull
        git push
        
    5、如果在本地更改代码及时上传到git工具，公司的文件就要更新...
       pull
