```
DOM : document Object Model
        文档 对象 模型
        整个页面文档

官方解释:
是W3c组织推荐的处理可扩展标志语言的标准编程接口，在网页上组织页面
(或文档)的对象被组织在一个树形结构中，用来表示文档中对象的标准模型
就称为DOM
简单点来说：
DOM通过document提供一些能够操作页面内容的属性或者方法，赋予开发者
操作页面的能力。
比如：document.getElementById('div1');
```
